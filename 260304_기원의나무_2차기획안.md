# 기원의 나무 (Tree of Origin) 2차 실무 기획안

**주제**: "기술은 변해도 인간은 변하지 않는다" (과거와 미래의 연결)
**핵심 장치**: 태블릿 PC(또는 키오스크), 메인 스크린(LED/빔), 조명 및 전식 연출 장치
**사용 AI**: 나노바나나(Nano Banana) API (텍스트/프롬프트 기반 초고속 이미지 생성)

## 1. 개요 및 기획 의도
1차 기획안의 복잡한 미디어 맵핑을 덜어내고, **네오전곡리안 프로젝트에서 기 구축된 '입력형 키오스크 인터페이스'와 '나노바나나 API 이미지 렌더링 파이프라인'을 재사용**하여 구축 효율성과 예산 절감을 극대화한 현실적인 2차 실무 기획안입니다.
관람객이 남긴 다짐과 텍스트가 실시간으로 고퀄리티 이미지 카드(지혜의 열매)로 치환되어, 거대한 메인 스크린(나무)에 주렁주렁 열리는 직관적이고 감동적인 인터랙션 아크를 제공합니다.

## 2. 5단계 진행 플로우 (User Journey)

### Step 1: 텍스트 입력 (참여)
*   **세팅**: 나무 앞(또는 입장 동선)에 비치된 태블릿 PC 사용.
*   **행동**: 관람객은 "2029년 연천 구석기 엑스포에 기대하는 점" 또는 "나의 ESG 실천 다짐(예: 일회용품 줄이기)"을 텍스트로 입력합니다.
*   **UI/UX**: 네오전곡리안 키보드 UI를 차용하여, 터치스크린 환경에서 남녀노소 누구나 쉽게 타이핑할 수 있도록 직관적인 환경을 제공합니다.

### Step 2: 실시간 이미지 생성 (나노바나나 API 연동)
*   **백엔드**: 관람객이 입력한 텍스트 데이터가 나노바나나 API를 통해 즉시 서버로 전송됩니다.
*   **프롬프팅 로직**: 입력된 한글 텍스트를 AI 모델에 최적화된 배경 프롬프트(예: 몽환적인 숲, 빛나는 이파리 무늬, 타로카드 스타일 테두리 등)와 결합하여 고품질의 **'지혜의 열매(카드)' 이미지 1장**으로 생성합니다.

### Step 3: 생성 결과물 개별 확인
*   **액션**: 약 3~5초의 생성 대기 시간 후, 태블릿 화면에 방금 만들어진 완성형 이미지 카드가 노출됩니다.
*   **UX**: 관람객은 자신이 쓴 다짐이 얼마나 아름다운 시각물로 탄생했는지 확인하며 개인적인 만족감과 몰입을 느낍니다.

### Step 4: 메인 스크린 전송 연출 (조명/전식 결합 연출)
*   **행동**: 화면 상의 `[기원의 나무에 달기(전송)]` 버튼을 스와이프 하거나 터치합니다.
*   **물리적 연출**: 전송되는 즉시, 태블릿 거치대에서부터 메인 스크린(스크린 상의 가상의 나무 조형물)을 향해 뻗어 있는 **LED 전식 라인(또는 바닥/천장의 간접 점멸 조명)이 촤르륵 빛 번짐(스윕) 효과**를 냅니다. 이는 디지털 데이터가 날아가는 듯한 생생한 감각을 시각과 청각적으로 제공합니다.

### Step 5: 데이터 전시 및 누적 (Tree of Data)
*   **화면 송출**: 번쩍이는 시각 연출 라인을 타고, 관람객의 카드가 메인 스크린 속 '기원의 나무' 나뭇가지 특정 위치에 안착하여 빛을 냅니다.
*   **전시 효과**: 수백·수천 명의 관람객이 남기고 간 이미지 카드들이 계속해서 가지 위에 쌓이고 모이면서, 축제 마지막 날에는 화면 전체가 찬란한 빛과 다짐들로 가득 찬 하나의 거대한 '데이터 나무' 아카이빙 월이 완성됩니다.
